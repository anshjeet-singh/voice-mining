# Learnings: youtube-script

Accumulated craft lessons applied on every run.

<!-- 2026-07-11 · job 270005 · client: Trent Kus -->
- The SOP is silent on regulated niches. Add a compliance gate step (run before the anti-pattern audit) for finance, health, and credit clients: no guaranteed amounts, timelines, or deletions; state the pricing MODEL never the number; proof only as [PROOF: ...] placeholders gated on a real screenshot; and any contrarian diagnosis line stays a spoken hook or body bridge, never an on-screen headline card. The universal copy-quality gates already apply, but a script writer needs them named in the skill so a regulated deliverable does not ship a claim that flags the channel.
- Add a "match the CTA to the traffic's awareness level" rule to Step 4. Cold, problem-aware YouTube traffic converts to a low-friction next step (free community, keyword lead magnet), not a booking call; a "book a call / verify me / look me up" ask belongs only on warmer, solution-aware placements and underperforms as the ask on broad organic. This mirrors the same fix already logged for reel-scripter.
- The skill offers bullets (strong-on-camera) or full draft (freezes without a script) but has no path for a founder who freezes on teleprompter AND chokes on long-form. Add a segmented long-form mode: build the video as standalone chaptered segments (hook, arcs, CTA), write the hook and CTA word for word, keep the body a beat scaffold with hero lines, and require a "freestyle from the beats, do not read verbatim, shoot in segments" recording note at the top. This resolves the same tension already promoted to a first-class WORD_FOR_WORD mode in breakout-video-writer.
- When youtube-script runs inside a client-facing deliverable (e.g. the client-onboarding-orchestrator), ship ONE final title, ONE thumbnail brief, and ONE CTA, not the "three CTA options, pick one" menu the SOP describes. The orchestrator's document rules forbid option lists and choice commentary in a document handed to the client; the options menu is an internal ideation pass only. Same carve-out already logged for reel-scripter.
