# Learnings: offer-extraction

Accumulated craft lessons applied on every run.

<!-- 2026-07-14 · job 420001 · client: Trent Kus -->
- Author the Low/Mid ticket rung of the Offers doc as a display-ready value stack (the same value-stack shape the paid Skool About page uses), not just prose, so the downstream Skool build can reproduce the offer verbatim instead of re-deriving it. Congruence across the ladder converts, and a mushy handoff forces a weak paid About page.

- Author the Low/Mid rung of the Offers doc as a display-ready value stack in the exact paid-Skool value-stack shape (hook headline, early-bird price, proof line, mechanism, ✅ items each ending in a ($ anchor), 🎁 bonuses with ($ values), 🏷️ Total Value vs price, Book A Call CTA, 🔥 ANNUAL PLAN block, 💥 JOIN NOW), so the downstream community build reproduces the offer verbatim instead of re-deriving a weaker one.
